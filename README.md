# Netflix Data Analysis

## Description
This project analyzes Netflix movies and TV shows dataset to understand trends, ratings, and content distribution.

## Dataset
- Source: Netflix Dataset
- Contains information about movies and TV shows

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Visualization
4. Basic Insights

## Results
- Movies are more than TV Shows
- Most content released after 2010
- TV-MA is the most common rating

## Tools Used
- Python
- Pandas
- Matplotlib

## Conclusion
Netflix content has grown rapidly in recent years with a focus on mature audiences.
